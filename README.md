# TP-INTEGRADOR-PROGRAMACION-2

LINK YOUTUBE: https://youtu.be/-mcQPVrm-p8

Este es nuestro proyecto DISPOSITIVOS IOT - CONFIGURACIONRED (JAVA - JDBC + MYSQL). Implementa un sistema de gestion de dispositivos IoT y sus configuraciones de red, desarrollado en java utilizando JDBC para la conexion a MYSQL.
El proyecto permite realizar operaciones CRUD ( CREAR,LEER, ACTUALIZAR, ELIMINAR),  manejo de conexiones transacciones y pruebas mediantes una clase de tst y un menu interactivo. Entre los dos integrantes nos pusimos a trabajar en conjunto y pudimos tener un gran avance, poder darle final a este trabajo.
